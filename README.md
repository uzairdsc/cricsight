# Wagon Plots

Password-protected Streamlit dashboard for cricket ball-by-ball visual analysis.

Build wagon wheels, wagon zones, and dismissal plots with filters for player, team, opposition, venue, phase, innings, and date range. Also supports batch plot generation and ZIP download for selected players.

## Data Sources
- CSV upload
- S3
- Local cache files

## Usage
1. Open the app in Streamlit
2. Load a dataset from the sidebar
3. Set filters and select the plot type
4. Generate plots individually or in batch
5. Download figures or the batch ZIP file
